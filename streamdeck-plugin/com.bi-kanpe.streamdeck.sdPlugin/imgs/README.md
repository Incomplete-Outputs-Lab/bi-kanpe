# StreamDeck Plugin Icons

This directory should contain the following icon files:

## Required Icons

### Action Icons
- `send-feedback.png` - Send Feedback action icon (144x144px)
- `send-feedback@2x.png` - Send Feedback action icon retina (288x288px)
- `react-to-latest.png` - React to Latest action default state (144x144px)
- `react-to-latest@2x.png` - React to Latest action default state retina (288x288px)
- `react-to-latest-alert.png` - React to Latest action alert state (144x144px)
- `react-to-latest-alert@2x.png` - React to Latest action alert state retina (288x288px)

### Plugin Icons
- `plugin-icon.png` - Plugin icon in StreamDeck store (256x256px)
- `plugin-icon@2x.png` - Plugin icon retina (512x512px)
- `category-icon.png` - Category icon (28x28px)
- `category-icon@2x.png` - Category icon retina (56x56px)

## Icon Design Guidelines

- Use simple, recognizable symbols
- High contrast for visibility on dark backgrounds
- Consistent style across all icons
- PNG format with transparency
- Use StreamDeck's color palette where appropriate

## Temporary Placeholders

Until proper icons are created, you can use simple colored squares or text-based placeholders.

### Creating Placeholder Icons

You can create simple placeholder icons using any image editor:
1. Create a canvas of the required size
2. Fill with a solid color
3. Add text or simple symbol
4. Export as PNG

### Recommended Tools

- Adobe Illustrator / Photoshop
- Figma
- Sketch
- GIMP (free)
- Inkscape (free)
