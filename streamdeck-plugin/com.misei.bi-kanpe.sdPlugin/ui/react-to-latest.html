Usage Guide
  • The button appearance changes when a new message arrives
  • Press to send a quick reaction to the latest message
  • Button returns to normal state after sending
Setup Instructions
  1. Launch Bi-Kanpe in Caster Mode
  2. Connect to the director's server
  3. Enable StreamDeck server in the app
  4. Enter the server address (default: localhost:9877)
  5. Choose your preferred reaction type
